---
title: animate()
description: The unified engine — spring physics or easing curves, one function.
---

`animate()` is what `spring()`, `enter()`, and `tween()` all call internally. Use it directly when you need easing curves.

The mode switches on the `ease` key:

```ts
import { animate } from '@tweens/tweens'

// Spring mode — physics, no fixed duration
animate(el, { x: 100 }, 'snappy')
animate(el, { x: 100 }, { stiffness: 300, damping: 28 })

// Easing mode — 'ease' key switches the engine
animate(el, { x: 100 }, { duration: 0.35, ease: 'easeOutExpo' })
animate(el, { x: 100 }, { duration: 0.4, ease: [0.19, 1, 0.22, 1] })
```

## Color animation

Animate `backgroundColor` and `color` between hex or `rgb()` / `rgba()` values.

<ColorDemo />

```ts
animate(el, { backgroundColor: '#3b82f6' }, { duration: 0.3, ease: 'easeOutCubic' })
animate(el, { color: 'rgb(255, 255, 255)' }, { duration: 0.3, ease: 'easeOutCubic' })
```

## Easing names

tweens includes Emil Kowalski's full easing palette plus CSS equivalents.

**ease-out** — use for anything entering or exiting the screen (most common):

```ts
'easeOutQuad'   // weak
'easeOutCubic'  // default — good for most UI
'easeOutQuart'
'easeOutQuint'
'easeOutExpo'   // strong — feels fast and decisive
'easeOutCirc'
```

**ease-in-out** — use for elements already on screen that need to move:

```ts
'easeInOutQuad'
'easeInOutCubic'
'easeInOutQuart'
'easeInOutQuint'
'easeInOutExpo'
'easeInOutCirc'
```

**Other:**

```ts
'ease'      // hover states and color transitions
'linear'    // constant speed — marquees, progress indicators only
'easeIn'    // avoid — slow start feels sluggish in UI
'easeOut'   // CSS ease-out equivalent
'easeInOut' // CSS ease-in-out equivalent
```

**Custom cubic-bezier** — pass a 4-value tuple directly:

```ts
animate(el, { x: 100 }, { duration: 0.4, ease: [0.19, 1, 0.22, 1] })
```

**Custom function:**

```ts
animate(el, { x: 100 }, { duration: 0.4, ease: (t) => t * t })
```

You can also import `cubicBezier` to build a reusable easing function:

```ts
import { cubicBezier } from '@tweens/tweens'

const myEase = cubicBezier(0.19, 1, 0.22, 1)
animate(el, { x: 100 }, { duration: 0.4, ease: myEase })
```

## Signature

```ts
animate(
  el: HTMLElement,
  props: AnimateProps,
  options?: AnimateOptions | SpringPreset,
): Promise<void>
```

## Shared options

Both modes support these:

```ts
animate(el, { x: 100 }, {
  duration: 0.4,
  ease: 'easeOutCubic',
  delay:      0.1,   // seconds before start
  repeat:     2,     // extra plays after first; -1 = infinite
  yoyo:       true,  // reverse on every other repeat
  onStart:    () => {},
  onUpdate:   (v) => {},
  onComplete: () => {},
})
```
