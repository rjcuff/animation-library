---
title: spring()
description: Animate to target values with spring physics. The most common call in any tweens project.
---

Give `spring()` an element, the values to animate to, and an optional config. It returns a `Promise<void>` that resolves when the spring settles.

```ts
import { spring } from '@tweens/tweens'

spring(el, { x: 100, scale: 1.03 }, 'snappy')
spring(el, { opacity: 0 }, { stiffness: 300, damping: 28 })
```

## Press feedback

Compress on press, spring back on release. `'stiff'` makes the compress feel instant — users perceive direct contact.

## Hover lift

Scale up and rise on hover, return when the cursor leaves. Move off mid-animation and the spring picks up from wherever it is — no snapping.

## Interruptible

Call `spring()` on an element already mid-animation. The new spring inherits the current velocity — no snap, no reset.

<InterruptDemo />

Hit Play repeatedly to see velocity carry through each reversal.

## Signature

```ts
spring(
  el: HTMLElement,
  props: AnimateProps,
  options?: SpringConfig | SpringPreset,
): Promise<void>
```

## Relative values

Prefix with `+=` or `-=` to animate relative to the current position:

```ts
spring(el, { x: '+=100' })     // add 100px to current x
spring(el, { x: '-=50' })      // subtract 50px
spring(el, { scale: '-=0.05' })
```
