---
title: spring()
description: Animate to target values using spring physics. The most common call in any tweens project.
---

`spring()` is the core function. Give it an element, the values you want to animate to, and an optional config. It returns a `Promise<void>` that resolves when the spring physically settles.

```ts
import { spring } from '@tweens/tweens'

spring(el, { x: 100, scale: 1.05 }, 'snappy')
spring(el, { opacity: 0 }, { stiffness: 300, damping: 20 })
```

## Press feedback

Squeeze down on press, spring back on release. The `'stiff'` preset makes the compress instant — users perceive it as direct physical contact.

## Hover lift

Scale up and rise when hovered, return when the cursor leaves. If the user moves off mid-animation, the spring picks up from current position and velocity — no jumping.

## Interruptible

Call `spring()` on an element that's already mid-animation. The new spring inherits the current velocity — there's no snap or reset.

<InterruptDemo />

Hit Play multiple times in a row to see velocity carry through each reversal.

## Signature

```ts
spring(
  el: HTMLElement,
  props: AnimateProps,
  options?: SpringConfig | SpringPreset,
): Promise<void>
```

## Relative values

Prefix with `+=` or `-=` to animate relative to the current position.

```ts
spring(el, { x: '+=100' })   // add 100px to current x
spring(el, { x: '-=50'  })   // subtract 50px from current x
spring(el, { scale: '-=0.05' })
```
