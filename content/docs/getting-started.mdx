## Install

```bash
npm install @tweens/tweens
```

## The model

Every tweens function takes a **DOM element**, the **properties to animate to**, and an optional **config**:

```ts
spring(element, { x: 100, opacity: 1 }, 'snappy')
```

tweens reads the element's current state and runs until the spring settles. No from-values, no duration guessing.

## In React

Use `useRef` to get the DOM element. Avoid `document.querySelector` — you won't have a stable ref across renders.

```tsx
'use client'
import { useRef, useEffect } from 'react'
import { spring, enter } from '@tweens/tweens'

export function Card() {
  const ref = useRef<HTMLDivElement>(null)

  useEffect(() => {
    enter(ref.current!, { y: 20, opacity: 0 }, 'gentle')
  }, [])

  return (
    <div
      ref={ref}
      className="opacity-0"
      onMouseEnter={() => spring(ref.current!, { y: -4, scale: 1.02 }, 'snappy')}
      onMouseLeave={() => spring(ref.current!, { y: 0, scale: 1 }, 'snappy')}
    />
  )
}
```

Add `className="opacity-0"` on elements that animate in. `enter()` reads the natural target state before snapping to the from-state, so it handles the pre-hidden case automatically.

## Common patterns

### Hover

```ts
el.addEventListener('mouseenter', () => spring(el, { y: -4, scale: 1.03 }, 'snappy'))
el.addEventListener('mouseleave', () => spring(el, { y: 0, scale: 1 }, 'snappy'))
```

Move off mid-animation and the spring picks up from current position and velocity — no snapping.

### Entrance

`enter()` snaps the element to a from-state, then springs it back to its natural position.

```ts
enter(el, { y: 16, opacity: 0 }, 'snappy')
```

### Sequencing

Spring functions return Promises, so you can chain with `async/await`:

```ts
async function revealContent() {
  await spring(heading, { y: 0, opacity: 1 }, 'snappy')
  await spring(body, { y: 0, opacity: 1 }, 'gentle')
  await cascade(cards, { y: 0, opacity: 1 }, 'snappy', 0.06)
}
```

## Presets

Four named presets cover most cases. Pass as the third argument:

| Preset | Feel |
|--------|------|
| `'snappy'` | Fast, no bounce — hover, focus, drag |
| `'gentle'` | Slow, smooth — entrances, reveals |
| `'bouncy'` | Subtle overshoot — drag-release, playful feedback |
| `'stiff'` | Instant compress — press feedback |

```ts
spring(el, { scale: 1.03 }, 'snappy')
enter(el, { opacity: 0, y: 12 }, 'gentle')
```

## Accessibility

tweens checks `prefers-reduced-motion` automatically. Under reduced motion, animations resolve instantly — no extra code needed.
