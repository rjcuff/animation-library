---
title: enter() / tween()
description: Entrance helpers. enter() springs from a state back to natural position. tween() animates between two explicit states.
---

## enter()

`enter()` snaps the element to a "from" state, then springs it back to wherever it currently sits. You describe where to come from — not where to go.

This is the go-to for entrance animations. Set `style={{ opacity: 0 }}` on the element so it starts invisible, then call `enter()` on mount.

<ToastDemo />

The toast enters from below, sits for a moment, then exits upward.

```ts
import { enter } from '@tweens/tweens'

// Slide up and fade in from below
enter(el, { y: 20, opacity: 0 }, 'snappy')

// Scale in from slightly small
enter(el, { scale: 0.94, opacity: 0 }, 'gentle')

// In React — call in useEffect
useEffect(() => {
  enter(ref.current!, { y: 16, opacity: 0 }, 'snappy')
}, [])
```

### Signature

```ts
enter(
  el: HTMLElement,
  fromProps: AnimateProps,
  options?: AnimateOptions | SpringPreset,
): Promise<void>
```

## tween()

`tween()` is the explicit version of `enter()`. You control both the starting state and the ending state.

```ts
import { tween } from '@tweens/tweens'

// Slide in from the left
tween(el, { x: -120, opacity: 0 }, { x: 0, opacity: 1 }, 'gentle')

// With easing instead of spring
tween(el, { opacity: 0 }, { opacity: 1 }, { duration: 0.3, ease: 'easeOut' })
```

### Signature

```ts
tween(
  el: HTMLElement,
  fromProps: AnimateProps,
  toProps: AnimateProps,
  options?: AnimateOptions | SpringPreset,
): Promise<void>
```

## Magnetic follow

Track cursor position by calling `spring()` on every `mousemove`. The spring config controls how tightly the element follows — `'snappy'` feels responsive, `'gentle'` feels sluggish.

<MagneticDemo />

```ts
const onMouseMove = (e) => {
  const rect = el.getBoundingClientRect()
  const dx = (e.clientX - rect.left - rect.width / 2) * 0.45
  const dy = (e.clientY - rect.top - rect.height / 2) * 0.45
  spring(el, { x: dx, y: dy }, 'snappy')
}
el.addEventListener('mouseleave', () =>
  spring(el, { x: 0, y: 0 }, 'gentle')
)
```
