---
title: enter() / tween()
description: Entrance helpers. enter() springs from a state back to natural position. tween() animates between two explicit states.
---

## enter()

`enter()` snaps the element to a from-state, then springs back to wherever it naturally sits. You describe where it comes from — not where it goes.

Add `className="opacity-0"` (or `style={{ opacity: 0 }}`) to pre-hide the element. `enter()` reads the natural target state before writing the from-state, so the pre-hidden opacity is handled automatically.

<ToastDemo />

```ts
import { enter } from '@tweens/tweens'

// Slide up and fade in
enter(el, { y: 20, opacity: 0 }, 'snappy')

// Scale in from slightly small
enter(el, { scale: 0.95, opacity: 0 }, 'gentle')

// In React
useEffect(() => {
  enter(ref.current!, { y: 16, opacity: 0 }, 'snappy')
}, [])
```

### Signature

```ts
enter(
  el: HTMLElement,
  fromProps: AnimateProps,
  options?: AnimateOptions | SpringPreset,
): Promise<void>
```

## tween()

`tween()` is the explicit version — you control both the start and end state.

```ts
import { tween } from '@tweens/tweens'

// Slide in from the left
tween(el, { x: -120, opacity: 0 }, { x: 0, opacity: 1 }, 'gentle')

// With easing
tween(el, { opacity: 0 }, { opacity: 1 }, { duration: 0.3, ease: 'easeOutCubic' })
```

### Signature

```ts
tween(
  el: HTMLElement,
  fromProps: AnimateProps,
  toProps: AnimateProps,
  options?: AnimateOptions | SpringPreset,
): Promise<void>
```

## Magnetic follow

Track cursor position by calling `spring()` on every `mousemove`. The spring config controls how tightly the element follows.

<MagneticDemo />

```ts
const onMouseMove = (e) => {
  const rect = el.getBoundingClientRect()
  const dx = (e.clientX - rect.left - rect.width / 2) * 0.45
  const dy = (e.clientY - rect.top - rect.height / 2) * 0.45
  spring(el, { x: dx, y: dy }, 'snappy')
}
el.addEventListener('mouseleave', () => spring(el, { x: 0, y: 0 }, 'gentle'))
```
